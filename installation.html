Install – the latest release of CUTE

You can use the CUTE unit testing framework with the Eclipse IDE or as a standalone solution.

By installing CUTE you agree to our license.

Please note that CUTE requires Java 8 to run. See the Frequently Asked Questions for instructions.

Install the CUTE Eclipse Plug-in

The CUTE plug-in can either be installed from the Eclipse Marketplace or by using our updatesite directly.

Using the Eclipse Marketplace

This is the easiest way to get CUTE. Choose Help > Eclipse Marketplace in the Eclipse menu and search for CUTE. This will install the CUTE plug-in and required CUTE header files.

Note: If you require to install optional CUTE features, please install them from the CUTE updatesite.

Using the CUTE Updatesite

In Eclipse, do the following:

  1. Choose Help > Install New Software in the Eclipse menu to start the Eclipse software installer.
  2. Click Add... to add the CUTE repository.
  3. Type (or copy) http://www.cute-test.com/updatesite into the Work with: text field and confirm with Enter.
  4. Choose at least the CUTE Plugin feature.
  5. Click Next. Eclipse will then resolve dependencies and show the install dialog.
  6. Read and accept the licences and click Finish if you want to continue installing CUTE and agree with the licences.
