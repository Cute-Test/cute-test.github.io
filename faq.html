FAQ – frequently asked questions

What is needed to run the CUTE plug-in?

The CUTE Eclipse plug-in depends on the following software:

  • Java JRE
  • Eclipse IDE with installed Eclipse C/C++ Development Tooling. Either download CDT combined with Eclipse ("Eclipse IDE for C/C++ Developers" from the Eclipse Download Page) or install into any Eclipse from the CDT updatesite (http://download.eclipse.org/releases/<Eclipse_Releasename>/).
  • Boost C++ Libraries (pre C++11): Install Boost C++ Libaries on your system or use the Boost Headers provided with the CUTE plug-in. Since C++11, Boost is no longer needed.
  • For CDT to be able to compile, you will also need to have an Eclipse compatible C++ compiler installed.

What are the supported platforms?

The CUTE plug-in works with Eclipse CDT Kepler releases (or newer). Framework version 2.0 was tested with the following compilers and operating systems:

  • Linux: gcc 4.7
  • Mac OSX: gcc 4.8 -std=c++11, clang (Xcode) with libc++ -std=c++11, gcc421 (Xcode) with Boost
  • Windows 7: MinGW with Boost, VS2012 NTP, VS2012, VS2010

Other combinations are possible. C++03 should be supported fine if you have boost, either installed or the headers that are packaged with the Eclipse plug-in.
